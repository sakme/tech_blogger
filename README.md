# Tech Blogger

Tech Blogger is a website built with JavaScript, Node JS, Express JS, a MySQL database, and several NPMs. A demo instance of the site is published [here](https://infinite-forest-94255.herokuapp.com).

## Table of Contents

- [Usage](#usage)
- [License](#license)
- [Questions](#questions)

## Usage

Any one can view posts on the homepage.  To create your own posts or to comment on others posts, click Login to sign in or to enroll.  Once logged in, you can click on posts on the homepage to comment.  On the Dashboard page, you will see all of your posts. Clicking on the post will allow you to edit or delete the post.
